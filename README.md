# SecureScan

Application de scan de sécurité pour dépôts GitHub (Semgrep, npm audit, TruffleHog). Saisie d’une URL → rapport avec score et vulnérabilités.

- **Front** : formulaire (URL), page rapport (score, grade, liste d’alertes).
- **Backend** : voir [README-BACKEND.md](./README-BACKEND.md).

`npm run dev` pour lancer le projet.
